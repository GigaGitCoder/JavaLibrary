# Quick Start

- Linux/macOS:

```bash
git clone --recurse-submodules https://github.com/GigaGitCoder/JavaLibrary

cd JavaLibrary

./setup-build.sh

./start.sh
```
`./stop.sh/` для выключения.
  
- Windows:
```cmd
git clone --recurse-submodules https://github.com/GigaGitCoder/JavaLibrary

cd JavaLibrary

start setup-build.ps1

start start.ps1
```
`start stop.ps1` для выключения.
